# Investigating Gender Stereotypes in LLM generated short stories

## Project Overview

This repository contains the code, notebooks, and data samples developed for my Master’s thesis in Cognitive Science, titled *“Once Upon a Bias: Investigating Gender Stereotypes in LLM-Generated Short Stories.”*
The project examines how gender stereotypes emerge in short stories generated by large language models, with a particular focus on the moderating role of literary genre. It is guided by the following research question:

### Research Question

> “How do gender stereotypes manifest in LLM-generated short stories, and to what extent does genre moderate these patterns?”

## How to run the code

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/Gender-stereotypes-in-LLM-generated-short-stories.git
cd Gender-stereotypes-in-LLM-generated-short-stories
```
### 2. Obtain an API Key

Obtain an API key from OpenAI (or another LLM provider):

https://platform.openai.com/api-keys

### 3. Install Dependencies

Install the required Python packages:

```bash
pip install -r requirements.txt
```

### 4. Run the Notebooks

The analysis pipeline is organized into four main stages:

#### Story Generation

Run:

- `story_hint_generation.ipynb`
- `story_generation.ipynb`

#### Story Quality Checks

Run:

- `blind_genre_prediction.ipynb`
- `adding_text_descriptors.ipynb`
- `calculating_average_text_descriptors.ipynb`
- `toxicity.ipynb`
- `story_similarity.ipynb`

#### Training Embeddings

Run:

- `training_word_embeddings.ipynb`
- `word_embedding_quality_checks.ipynb`

#### Gender Bias Analysis

Run:

- `extract_and_classify_names.ipynb`
- `representational_bias.ipynb`
- `associational_bias.ipynb`
- `associational_bias_plotting.ipynb`
- `associational_bias_modeling.Rmd`
- `SRL_extraction.ipynb`
- `SRL_cleaning.ipynb`
- `relational_bias_plotting.ipynb`
- `relational_bias_modeling.Rmd`
