# Thesis: Investigating Gender Stereotypes in LLM generated short stories

### Project Overview

This repository contains the code, notebooks, and data samples developed for my Master’s thesis in Cognitive Science, titled “Once Upon a Bias: Investigating Gender Stereotypes in LLM-Generated Short Stories.”
The project examines how gender stereotypes emerge in short stories generated by large language models, with a particular focus on the moderating role of literary genre. It is guided by the following research question:

“How do gender stereotypes manifest in LLM-generated short stories, and to what extent does genre moderate these patterns?”

### Structure of the repository
| Folder              | Description |
|---------------------------|-------------|
| `data_sample/`            | Contains sample of the generated stories |
| `notebooks/`              | Contains all source code |
| `results/`                | Contains all plots, models and tables |
